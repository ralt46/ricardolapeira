Memory Is a Fragile Thing

If you don’t write it down, you will probably forget it. For some reason, we tend to think otherwise.

Sometimes I tell myself a lie: “this is so important that I won’t forget it.” Deep down, I know I am going to forget it if I’m not careful. Not in a ‘I will never remind it kind of way,’ but rather in a way that makes me loose focus. That’s why I’ve started writing down everything that truly matters to me.

It kind of makes me think of the movie Memento. It’s main character has to rebuild his memory every day. He does so by having props that will help him access very basic stuff like who he is and what he’s doing.

This might sound a bit dramatic, but it isn’t. Write it down, or otherwise you will forget it. Memory is a fragile thing.
